Enums:
```typescript
import { pgTable, pgEnum, serial, text, integer, boolean, timestamp, decimal, varchar, uniqueIndex } from "drizzle-orm/pg-core";
import { relations } from "drizzle-orm";

export const userRoleEnum = pgEnum("user_role", ["staff", "manager", "guest"]);
export const slipStatusEnum = pgEnum("slip_status", ["available", "occupied", "departing_today", "maintenance"]);
export const slipSizeEnum = pgEnum("slip_size", ["small", "medium", "large"]);
export const stayStatusEnum = pgEnum("stay_status", ["reserved", "checked_in", "checked_out"]);
export const fuelTypeEnum = pgEnum("fuel_type", ["diesel", "gas"]);
export const amenityTypeEnum = pgEnum("amenity_type", ["shower", "fuel", "shore_power", "pump_out", "laundry"]);
```

Tables (use `serial("id").primaryKey()` for all IDs):

**users**: id, name (varchar 100), email (varchar 255, unique), passwordHash (text), role (userRoleEnum), phone (varchar 20, nullable), createdAt (timestamp, defaultNow)

**slips**: id, name (varchar 10, unique, e.g. "T-1"), maxLoa (integer, feet), maxBeam (integer, feet), waterDepth (integer, feet), size (slipSizeEnum), status (slipStatusEnum, default "available"), hasShorepower (boolean, default true), createdAt (timestamp, defaultNow)

**guests**: id, name (varchar 100), email (varchar 255, nullable), phone (varchar 20, nullable), vesselName (varchar 100), vesselLoa (integer, feet), vesselBeam (integer, feet), vesselDraft (integer, feet), notes (text, nullable), createdAt (timestamp, defaultNow)

**stays**: id, guestId (integer, references guests.id), slipId (integer, references slips.id), checkIn (timestamp), checkOut (timestamp, nullable), expectedDeparture (timestamp), status (stayStatusEnum, default "reserved"), gateCode (varchar 6), wifiPassword (varchar 50), showerTokens (integer, default 3), showerTokensUsed (integer, default 0), nightlyRate (integer, cents), isPreBooked (boolean, default false), createdAt (timestamp, defaultNow)

**amenityUsage**: id, stayId (integer, references stays.id), type (amenityTypeEnum), quantity (decimal 10,2, default 1), unitPrice (integer, cents), totalAmount (integer, cents), fuelType (fuelTypeEnum, nullable), notes (text, nullable), createdAt (timestamp, defaultNow)

**charges**: id, stayId (integer, references stays.id), description (varchar 200), category (varchar 50), amount (integer, cents), createdAt (timestamp, defaultNow)

**pricing**: id, category (varchar 50), name (varchar 100), rate (integer, cents), unit (varchar 20), createdAt (timestamp, defaultNow), updatedAt (timestamp, defaultNow)

Define Drizzle relations for all foreign keys (guests->stays, slips->stays, stays->amenityUsage, stays->charges).

3. Create `src/lib/session.ts`:

```typescript
import { getIronSession } from "iron-session";
import { cookies } from "next/headers";

export interface SessionData {
  userId: number;
  role: "staff" | "manager" | "guest";
  name: string;
}

const sessionOptions = {
  password: process.env.SESSION_SECRET!,
  cookieName: "harborpass-session",
  cookieOptions: {
    secure: process.env.NODE_ENV === "production",
    httpOnly: true,
    sameSite: "lax" as const,
    maxAge: 60 * 60 * 24 * 7, // 7 days
  },
};

export async function getSession() {
  return getIronSession<SessionData>(await cookies(), sessionOptions);
}
```
