Add three new query functions to src/lib/queries.ts:

1. `getOccupancyStats()` — Count slips by status. Return: `{ occupied: number, available: number, departingToday: number, maintenance: number, total: number }`. Query: `SELECT status, COUNT(*) FROM slips GROUP BY status`. Total transient slips = 20. Calculate percentages in the component.

2. `getRevenueBreakdown()` — Sum charges by category for current month. Return: `{ categories: { category: string, total: number }[], grandTotal: number }`. Query: `SELECT category, SUM(amount) FROM charges WHERE created_at >= startOfMonth GROUP BY category ORDER BY SUM(amount) DESC`. Use `startOfMonth(new Date())` from date-fns. Import `sql` from drizzle-orm for raw aggregation: `db.select({ category: charges.category, total: sql<number>\`sum(${charges.amount})\` }).from(charges).where(gte(charges.createdAt, startOfMonth(new Date()))).groupBy(charges.category)`.

3. `getGuestHistory()` — Fetch all stays (past and current) with guest info and total charges. Return array of `{ stayId, guestName, vesselName, vesselLoa, slipName, checkIn, checkOut, status, totalCharges, stayCount, charges: { description, category, amount, createdAt }[] }`. Query: Use db.query.stays.findMany with guest and slip and charges relations. Then compute stayCount per guest (group by guestId, count stays). Sort by checkIn DESC. A guest with stayCount > 1 is a repeat visitor.
