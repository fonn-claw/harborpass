- `getBoardData()`: Query all active stays (status "checked_in" OR status "reserved" with checkIn today) using `db.query.stays.findMany()` with `with: { guest: true, slip: true, amenityUsages: true }`. Use date-fns `startOfDay`/`endOfDay` for today range. Split results into three arrays:
  - `arriving`: status "reserved" AND checkIn within today
  - `departingToday`: status "checked_in" AND expectedDeparture within today

- `getAvailableSlips()`: Query slips where status is "available", ordered by name
- Export TypeScript types for the return values: `BoardData`, `SlipWithStay`

Use `or`, `eq`, `and`, `gte`, `lte` from drizzle-orm. Import date-fns for date range calculation.

IMPORTANT: Use the Neon HTTP driver pattern already established (import db from "@/db"). Do NOT attempt transactions — the Neon HTTP driver doesn't support them.
