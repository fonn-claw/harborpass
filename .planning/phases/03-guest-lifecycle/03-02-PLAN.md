1. Add `getSettlementData` query to `src/lib/queries.ts`:
   - Takes stayId (number)
   - Returns stay with guest, slip, and charges (ordered by createdAt asc) using db.query.stays.findFirst with `with:` relations
   - Also compute derived fields in the return: nightCount using differenceInCalendarDays(new Date(), stay.checkIn), and totalAmount as sum of all charges.amount
   - IMPORTANT: Do NOT recalculate nightly charges. The charges table already contains nightly charge records from the seed. Settlement total = SUM of ALL charges.amount for the stay.
   - Return type should include the stay with relations plus nightCount and totalAmount

2. Add `settleAccount` server action to `src/app/board/actions.ts`:
   - Takes stayId (number)
   - Step 1: Get the stay to find slipId: `const [stay] = await db.select().from(stays).where(eq(stays.id, stayId)).limit(1)`
   - Step 2: Update stay: set status="checked_out", checkOut=new Date() — `db.update(stays).set({ status: "checked_out", checkOut: new Date() }).where(eq(stays.id, stayId))`
   - Step 3: Release slip: if stay.slipId exists, `db.update(slips).set({ status: "available" }).where(eq(slips.id, stay.slipId))`
   - Step 4: revalidatePath("/board")
   - Return { success: boolean, error?: string }
   - Wrap in try/catch
   - Import: stays, slips from schema, eq from drizzle-orm
  </action>
