  <action>
Add `getGuestPortalData` to `src/lib/queries.ts`:

1. Takes userId (number)
2. Step 1: Get user email — `const [user] = await db.select().from(users).where(eq(users.id, userId)).limit(1)`. Return null if no user.
3. Step 2: Find guest by email — `const [guest] = await db.select().from(guests).where(eq(guests.email, user.email)).limit(1)`. Return null if no guest found. IMPORTANT: handle the case where guest has no email or user email doesn't match any guest.
4. Step 3: Find active stay with relations — `db.query.stays.findFirst({ where: and(eq(stays.guestId, guest.id), eq(stays.status, "checked_in")), with: { slip: true, amenityUsages: { orderBy: (au, { desc }) => [desc(au.createdAt)] }, charges: { orderBy: (c, { asc }) => [asc(c.createdAt)] } } })`
5. Return: `stay ? { guest, ...stay } : null` — this flattens the guest info alongside the stay data
6. Import: users, guests, stays from schema. eq, and from drizzle-orm.
